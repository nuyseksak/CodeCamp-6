1.ค้นหาข้อมูลว่า <¡DOCTYPE> มีหน้าที่อะไร?

Doctype : เป็น tag ที่ไว้บอกบราวเซอร์ว่า เอกสารที่กำลังจะเปิดอยู่เป็นมาตรฐานแบบไหน

2.ถ้าอยากให้มี tooltip ใน tag นั้นต้องใช้ attribute ใด?

Tooltip : ใช้ attribute title="ข้อความ"

3.แท็ก <b>,<i> ต่างจาก <Strong>,<em> หรือไม่อย่างไร?

<b>,<Strong> แสดงผลเป็นตัวหนาเหมือนกัน แต่ใช้ไม่เหมื่อนกัน
<b> element เป็นอักษรตัวหนา (bold) ใช้เน้นข้อความ
<Strong> element ใช้เน้นและขยายข้อความในสิ่งที่สำคัญมากๆ

<i>,<em> แสดงผลเป็นตัวเอียงเหมือนกัน แต่ใช้ไม่เหมื่อนกัน
<i> element เป็นอักษรตัวเอียง (italic) ใช้เป็นข้อความตัวเลือก เช่น ภาษาต่างประเทศ, ศัพท์ทางเทคนิค
<em> element ใช้ขยายข้อความ เช่น การอ่านออกเสียงที่ต่างกัน
