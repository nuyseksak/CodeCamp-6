CSS Syntax

CSS Syntax ประกอบไปด้วย selector และ Declaration Block

Selector คือ html element ที่ต้องการที่จะใส่ style

Declaration block คือ การบอกลักษณะ style ในแต่ละ declaration block จะถูกขั้นด้วยเครื่องหมาย(semicolons) ;

ในแต่ละ declaration block จะประกอบไปด้วย property และ value จะถูกกขั้นด้วย(colon) :

Property คือ คุณสมบัติ

Value คือ คือค่าที่ระบุ

Inline CSS

Inline CSS จะใช้ได้เฉพาะ element นั้น element เดียว

กำหนด css ได้ที่ attribute style ใน element

        <p style="color:red;">This is a paragraph.</p>

External CSS

External CSS นำเข้าไฟล์จากภายนอกเข้ามาใน html page

โดยนำเข้าได้จาก <link> element อยู่ถายใน <head>

        <head>
            <link rel="stylesheet" type="text/css" href="mystyle.css">
        </head>

Internal CSS

Internal CSS จะใช้ได้เฉพาะ html page นั้น page เดียว

กำหนด css ได้ที่ <style> element อยู่ถายใน <head>

        <head>
            <style>
                h1 {
                    color: maroon;
                    margin-left: 40px;
                }
            </style>
        </head>

ID Selectors

คือการเลือก html element โดยระบุ id

การกำหนดชื่อของ element id โดยจะมีได้แค่ element เดียวเท่านนั้นในหนึ่ง html page

การเลือก html element โดยระบุ id ต้องนำหน้าด้วยเครื่องหมาย hash (#) ตามด้วยชื่อ id

        #elementid {
            text-align: center;
            color: red;
          }

Class Selectors

คือการเลือก html element โดยระบุ class ใน attribute class ของ element

การกำหนดชื่อของ elemnet โดยสามารถมีได้หลาย element

การเลือก html element โดยระบุ class ต้องนำหน้าด้วยเครื่องหมาย period (.) ตามด้วยชื่อ class

        .classname{
            text-align: center;
            color: red;
          }

Pseudo-elements Selectors

กำหนดเฉพาะบางส่วนของ element

  • ::after เพิ่มข้อความหลังแท็ก
  • ::before เพิ่มข้อความก่อนแท็ก
  • ::first-letter เลือกตัวแรกของแท็ก
  • ::first-line เลือกแถวแรกของแท็ก
  • ::selection ครอบตัวหนังสือ

กำหนด element ตามด้วยเครื่องหมาย :: Pseudo-elements

        p::first-line {
            color: #ff0000;
            font-variant: small-caps;
          }

Pseudo-class Selectors

เลือก element เฉพาะบาง attribute

  • :active เลือกเฉพาะลิงค์ที่ทำงาน
  • :checked เลือกเฉพาะช่องที่ถูกเช็ค
  • :disabled เลือกเฉพาะช่องที่ไม่ทำงาน
  • :focus เลือก input element ที่ถูกคลิก
  • :hover เลือกเฉพาะบล็อกที่เมาส์เลื่อนโดน
  • :link เลือกทุกลิงค์ที่มี
  • :required เลือกเฉพาะช่องที่มี required
  • :target เลือกเฉพาะ target link

กำหนด element ตามด้วยเครื่่องหมาย : Pseudo-class

        selector:pseudo-class {
            property:value;
          }

Combinator Selectors

การกำหนดเหงื่อนไขระหว่าง 2 selector

  • Descendant selector (space)
  • Child selector (>)
  • Adjacent sibling selector (+)
  • General sibling selector (~)

ตามตัวอย่างจะเลือก tag <p> ทั้งหมดที่อยู่ใน <div> 

        div p {
            background-color: yellow;
          }

Attribute Selectors

การกำหนดเลือก attribute ใน element

ตามตัวอย่าง จะเลือกเฉพาะ element ที่มี attribute target="_blank"

        a[target="_blank"] {
            background-color: yellow;
          }

Descendant selector (space)

การเลือก element ทั้งหมดที่อยู่ภายใน base element

ตามตัวอย่างจะเลือก tag <p> ทั้งหมดที่อยู่ใน <div> 

        div p {
            background-color: yellow;
        }

Child selector (>)

การเลือก element ทั้งหมดที่เป็นลูกของ base element

ตามตัวอย่างจะเลือก tag <p> ทั้งหมดที่เป็นลูกของ <div> ทั้งหมด

        div > p {
            background-color: yellow;
          }

adjacent sibling selector (+)

การเลือก element ที่อยู่ถัดจาก base element

ตามตัวอย่างจะเลือก tag <p> ที่อยู่ถัดจาก <div> 

        div + p {
            background-color: yellow;
          }

general sibling selector (~)

การเลือก element ทั้งหมดที่อยู่ถัดจาก base element

ตามตัวอย่างจะเลือก tag <p> ทั้งหมดที่อยู่ถัดจาก <div> 

        div ~ p {
            background-color: yellow;
          }
