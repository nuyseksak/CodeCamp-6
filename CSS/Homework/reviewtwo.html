property และ value คืออะไรแตกต่างกันอย่างไร?

property คือ คุณสมบัติของ element
value คือ ลักษณะหรือค่าของ property นั้นๆว่าจะให้แสดงออกมาอย่างไร

property ที่สอนใน precamp มีกี่กลุ่มอะไรบ้าง?

มี property 5 กลุ่มได้แก่
Property for content :color , size , font , text, list-style-type, Height, Width, background
property for border :border, table
property for padding : padding (top right buttom left)
property for margin :margin (top right buttom left)
property for element :display, Height, Width, Position, Float, Align, Flex, Grid

value unit มีกี่ประเภทอะไรบ้างแต่ละ ประเภทคืออะไร?

CSS Units มีสองประเภทได้แก่
Absolute Lengths

Unit Description
cm centimeters
mm millimeters
in inches (1in = 96px = 2.54cm)
px * pixels (1px = 1/96th of 1in)
pt points (1pt = 1/72 of 1in)
pc picas (1pc = 12 pt)

Relative Lengths
Unit Description
em Relative to the font-size of the element (2em means 2 times the size of the current font)
ex Relative to the x-height of the current font (rarely used)
ch Relative to the width of the "0" (zero)
rem Relative to font-size of the root element
vw Relative to 1% of the width of the viewport*
vh Relative to 1% of the height of the viewport*
vmin Relative to 1% of viewport's* smaller dimension
vmax Relative to 1% of viewport's* larger dimension
% Relative to the parent element
